creative-nights.com

This website has very interesting animations while managing to keep a sleek and simple design. Scrolling reveals website elements in a dynamic fashion. For example, text is triggered by scrolling which makes the text appear more organic. Furthermore, as the user moves the mouse around, the cursor area reveals and hides the animation, video, or picture in the background. It is very much like a parallax effect but more versatile. One issue with this site though is that it is designed mainly for mobile users. For example, if you shrink the browser window size, the website asks you to "rotate your device" even though you may be on a desktop. Nevertheless, the rest of the site is competently constructed and the different effects make exploring the website more visually interesting.
