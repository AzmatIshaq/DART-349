This website is very "in your face" with its user experience while at the same time managing to maintain satisfactory usability. It utilizes windows to display animations and information. The windows allow the website to remain organized while a lot of things are happening on screen. It is an ambitious display of what a a non-standard user experience on a website can be like. However, one can argue that, in terms of good user experience, there is too much going on. For example, someone without a lot of website navigation experience may struggle to manage all the different windows and animations. Nevertheless, it is not too overly complex and one can familiarize themselves with the structure fairly easily. One area that can definitely be improved though is the slowly rotating menu animation. If it responded to mouse drags it would have been drastically more usable.
